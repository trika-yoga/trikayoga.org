::: stanza

ॐ {sans big}

auṃ  {trans}

वैदिकप्रणव {sans}

vaidikapraṇava {trans}

Vedic Pranava, the primordial sound of the Universe

:::

::: stanza

ह्रीं {sans big}

hrīṃ {trans}

शाक्तप्रणव {sans}

śāktapraṇava {trans}

Shakta Pranava


:::
::: stanza


हूं {sans big}

hūṃ {trans}

शैवप्रणव / कूर्चबीज {sans}

śaivapraṇava / kūrcabīja {trans}

Shaiva pranava or Kurcha Bija


:::
::: stanza

सौः {sans big}

sauḥ {trans}

प्रासादबीज / हृदयबीज {sans}

prāsādabīja / hṛdayabīja {trans}

Prasada bija or Hridaya bija

:::
::: stanza

अहं {sans big}

ahaṃ {trans}

सृष्टिबीज {sans}

sṛṣṭibīja {trans}

Srishti bija

:::
::: stanza

महा / मह अ {sans big}

mahā / maha a {trans}

संहरबीज {sans}

saṃharabīja {trans}

Samhara bija

:::
